# Todo App

## This a todo app created using `svelte js`

<hr>

<h3>Run this file on your device</h3>

- Clone Repo:

```bash
$ git clone https://github.com/kishcodes13/Todo-App
```

- Go to project directory and install dependencies:

```bash
cd Todo-App && npm install
```

- Run this by

```bash
npm run dev
```
